# NaturalDisaster
“Does the flap of a butterfly’s wings in Brazil set off a tornado in Texas?”  Natural Disaster is a point-and-click-inspired adventure game set in a View-Master. Explore a world in scenes and sow the seeds of disaster by virtue of the butterfly effect!
